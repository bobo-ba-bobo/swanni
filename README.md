# swanni & bobo

Bucket list for swanni & bobo - to finish all of them till grey and old.

우리 둘만 로그인해서 함께 채워가는 공간.

- **버킷리스트** (`/do`) — 함께 하고 싶은 것
- **먹킷리스트** (`/eat`) — 같이 먹고 싶은 것
- **노트** (`/notes`) — 우리 이야기
- **끝낸거** (`/done`) — 이미 이뤄낸 것들

스택: Next.js 14 · Supabase (인증 · DB · 실시간) · Vercel.
하늘하늘한 파스텔 글래스 테마.

## 시작하기

셋업은 [`SETUP.md`](./SETUP.md) 참고 — Supabase 프로젝트 생성, 계정 2개 만들기, Vercel 배포까지 단계별 안내.

```bash
cp .env.local.example .env.local   # Supabase URL / anon key 채우기
npm install
npm run dev
```
